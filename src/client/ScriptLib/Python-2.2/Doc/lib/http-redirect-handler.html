11.5.4 HTTPRedirectHandler Objects

Note: 303 redirection is not supported by this version of urllib2.

http_error_301(req, fp, code, msg, hdrs)
Redirect to the Location: URL. This method is called by the parent OpenerDirector when getting an HTTP permanent-redirect response.

http_error_302(req, fp, code, msg, hdrs)
The same as http_error_301(), but called for the temporary-redirect response.

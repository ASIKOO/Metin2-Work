3.3 Example

Here is a more complicated example from Modules/Setup.dist: 

GMP=/ufs/guido/src/gmp
mpz mpzmodule.c -I$(GMP) $(GMP)/libgmp.a

which could also be written as: 

mpz mpzmodule.c -I$(GMP) -L$(GMP) -lgmp

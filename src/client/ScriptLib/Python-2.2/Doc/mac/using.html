1. Using Python on the Macintosh

Using Python on the Macintosh can seem like something completely different than using it on a Unix-like or Windows system. Most of the Python documentation, both the ``official'' documentation and published books, describe only how Python is used on these systems, causing confusion for the new user of MacPython. This chapter gives a brief introduction to the specifics of using Python on a Macintosh.

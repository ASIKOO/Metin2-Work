1.6.1 Using the ``Python Interactive'' window

Use this window like you would the PythonInterpreter, except that you cannot use the ``Drag and drop'' method above. Instead, dropping a script onto the Python IDE icon will open the file in a separate script window (which you can then execute manually - see section 1.6.3).

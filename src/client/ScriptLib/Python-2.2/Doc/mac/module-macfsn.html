4.7 macfsn -- NavServices calls

Availability: Macintosh.

macfsn contains wrapper functions that have the same API as the macfs StandardFile calls, but are implemented with Navigation Services through the Nav module. Importing it will replace the methods in macfs with these, if Navigation Services is available on your machine.
